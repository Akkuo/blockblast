# Block Blast 量產規格：介面與體驗 (Art Pro UI/UX)

## 一、 介面配置 (UI Layout)
* **命名規範**：`ui_btn_pause.png`, `ui_panel_score.png`, `ui_icon_coin.png`
* **佈局分配**：
  - **Top (0-20%)**：分數面板、目標收集框、暫停按鈕。
  - **Center (20-70%)**：8x8 遊戲網格區。必須置中。
  - **Bottom (70-100%)**：底盤區（Spawn Area），放置 3 個候選方塊。下方為道具按鈕列。

## 二、 切圖倍率與規範 (Slicing)
* **九宮格切圖 (9-Slice)**：所有彈出視窗底板、進度條底框，必須提供適合九宮格拉伸的原始檔，標示清楚拉伸區域。
* **格式**：PNG-8 / PNG-24。

## 三、 交互動態設計 (Interaction Motion)
* **按下回饋**：所有按鈕點擊時必須有縮小 0.9x 的彈性動畫。
* **拖曳狀態**：方塊被手指按住並拖起時，方塊需放大至 1.1x，並上浮 (Y軸偏移)，底部增加透明陰影，解決手指遮擋視線的問題。
